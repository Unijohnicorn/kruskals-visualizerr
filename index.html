Comprehensive Guide & CLO Review
Overview
Kruskal's algorithm finds the Minimum Spanning Tree (MST) for a weighted undirected graph by always picking the smallest available edge that does not create a cycle. It continues until all vertices are connected.
Worked Example
Build steps to see the narrated example. Each step highlights the current greedy choice and the state of the MST components.
Key Concepts
• Greedy-choice property: Selecting the local minimum weight edge ensures no better global solution exists.
• Sorting by Weight: Pre-sorting edges is the foundation that enables a simple linear scan of candidate edges.
Greedy Paradigm (CLO-3)
Visualize each greedy choice and compare how selecting the minimum-weight edge at every step (local optimization) builds the global minimum tree cost without needing to backtrack.
